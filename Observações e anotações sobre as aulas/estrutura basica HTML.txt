No HTML a tag "alt = o valor"

Ex: exemplo de foto.

OBS: no inicio do codigo do HTML usa se a tap "lang" para dizer que aquele site foi feito em um determinado idioma.

Ex: 
<html lang="pt-br">
	<head>




	<head>
	<body>


      </body>
</html>

OBS 2: sempre usar a tag "meta charset='...'", ela é usada quando o site espera palavras com acentuação grafica.

Ex:
<html lang="pt-br">
	<head>
		<meta charset="UTF-8">


	<head>
	<body>


      </body>
</html>

OBS: Usar tmb a seguinte tag: <meta name="viewport" content="width=device-width,initial-scale=1.0">

 <html lang="pt-br">
	<head>
		<meta charset="UTF-8">
		<meta name="viewport" 
		content="width=device-width,initial-scale=1.0">
		<title></title>


	<head>
	<body>


      </body>
</html>
